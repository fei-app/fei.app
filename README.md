# OpenFEI #

<img src="https://gitlab.com/fei.app/fei.app/-/raw/f94216ed1b3bf80f180d9ff1317e9e197cf5d6ba/Android/app/src/main/ic_launcher-playstore.png" width="200"/> 

Meu cliente alternativo para acessar o portal do aluno do Centro Universitário FEI , com funcionalidades extras ao aplicativo oficial.

✅ FUNCIONALIDADES:
1. Interface Material 3 Expressive;
2. Amigável para tablets;
3. Notificações quando saem novas notas;
4. Notificações quando o horário de aulas é alterado;
5. Visualização de notas em tabela;
7. Tema escuro;
10. Cache offline das notas, calendário e horário de aula;
11. Resumos com IA (ChatGPT, Claude, Perplexity) (em desenvolvimento, será integrado resumo da matéria para cada prova no futuro);
12. Todas as funções do site....

Bom proveito!
